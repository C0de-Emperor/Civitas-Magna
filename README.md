# Consignes pour ouvrir le projet

* Installer Unity Hub
* Ouvrir Unity Hub
     * Installer la version de unity 6000.2.6f2
* Cloner ce dépot
* Via Unity Hub cliquer sur `Add > Project from disk`
    * Ouvrir le dépot cloné
* Lancer le projet

# Consignes d'installation

* Se rendre sur ce dépot github : [Civitas-Magna-Builds](https://github.com/ggglllkkk/Civitas-Magna-Builds) 
     * Cloner le dépot
* Executer le fichier "Civitas Magna.exe"
